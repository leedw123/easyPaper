# easyPaper

> 중소병원·의원을 위한 **무료 업무 서식 인쇄 프로그램**

별도의 문서 편집 프로그램 없이 진료 동의서, 서지, 각종 양식 등  
**1회성 인쇄 서식을 만들고 바로 출력**할 수 있는 업무용 프로그램입니다.

서버 1대에 설치하면 같은 네트워크의 여러 PC에서 **동시 접속**하여 사용할 수 있습니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 서식 작성 | 텍스트·체크박스·드롭다운·날짜·이미지 등 다양한 입력 위젯 |
| 1회성 인쇄 | A4 규격 즉시 출력. 여러 페이지 자동 분할 |
| 다중 사용자 | 여러 PC에서 동시 접속 사용 |
| 계정 관리 | 관리자 / 일반 사용자 권한 분리 |
| DB 연동 (선택) | SQLite · Oracle · MSSQL 연동으로 데이터 자동 채우기 |
| 자동 업데이트 | 서버에 새 버전을 올리면 접속 시 자동 배포 |
| 시스템 로그 | 누가 언제 어떤 작업을 했는지 이력 조회 |

---

## 사용 대상

- 의원 / 중소병원
- 진료 동의서, 입원 서지, 각종 안내 양식을 반복 출력하는 환경
- 별도 문서 프로그램 없이 서식을 통일하고 싶은 곳

---

## 구성 파일

| 파일 | 설명 |
|------|------|
| `easyPaperServer.exe` | 서버 프로그램. 데이터 저장·관리. 항상 켜두는 PC에 설치 |
| `easyPaper.exe` | 클라이언트. 사용자 PC에 복사하여 사용 |
| `easyPaperUpdater.exe` | 자동 업데이트 처리기. 클라이언트와 같은 폴더에 위치 |

---

## 설치 방법

### 서버 (관리자 1회 설치)

```
1. easyPaperServer.exe 를 서버 PC의 원하는 폴더에 복사
2. easyPaperServer.exe 실행
3. 콘솔에 표시된 클라이언트 연결 주소 확인
   예) http://192.168.1.10:8000
4. Windows 방화벽에서 8000번 포트 허용
```

### 클라이언트 (사용자 PC)

```
1. easyPaper.exe + easyPaperUpdater.exe 를 같은 폴더에 복사
2. easyPaper.exe 실행
3. 서버 주소 입력 (최초 1회)
4. 초기 계정으로 로그인
   아이디: admin  /  비밀번호: 1234
```

> 로그인 후 반드시 비밀번호를 변경하세요.

---

## DB 연동

DB 연동은 **선택 사항**입니다.  
연동하지 않아도 서식 작성·저장·인쇄·계정 관리는 모두 정상 동작합니다.

연동 시 검색창 → 데이터 자동 조회 → 서식 칸 자동 채우기 기능이 활성화됩니다.
